# Checkin Gabs 🥗

Plataforma integral diseñada para nutricionistas que buscan optimizar el seguimiento de pacientes mediante la digitalización de datos y la gamificación del progreso.

## 🚀 Descripción
Checkin Gabs facilita la gestión clínica al centralizar el historial del paciente, el seguimiento de medidas corporales y la evolución nutricional. La plataforma convierte métricas de salud en una experiencia gamificada para aumentar la adherencia del paciente al plan propuesto.

## 🛠 Tecnologías Utilizadas
Este proyecto está construido con un stack moderno enfocado en rendimiento y escalabilidad:

- **Frontend:** [Next.js](https://nextjs.org/), [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
- **Plataforma Móvil:** [React Native](https://reactnative.dev/)

## 📂 Estructura del Proyecto


* `src/components`: Componentes UI reutilizables (inputs, botones, cards).
* `src/hooks`: Lógica de estado personalizada para el seguimiento de pacientes.
* `src/app`: Rutas de la aplicación web.
* `src/services`: Conexión con la API y manejo de datos.
